About diagrams.free

diagrams.free is a browser-based whiteboard for sketching hand-drawn style diagrams. Draw in your browser, save locally, and export your work.

Open source

diagrams.free is built on Excalidraw, an open-source project released under the MIT License. We thank the Excalidraw team and contributors for their work. This deployment is maintained independently and is not affiliated with excalidraw.com or Excalidraw+.

Source code for this fork: konashevich/diagrams-free

File format

diagrams.free uses the .excalidraw file format for compatibility with the Excalidraw ecosystem.
